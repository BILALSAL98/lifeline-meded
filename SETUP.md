npm install

# Open http://localhost:3000

## Enable Real AI (Claude)

To get real AI responses in the Tutor and Recall features:

1. Get a free API key at [console.anthropic.com](https://console.anthropic.com)
2. Create a `.env.local` file in your project root:

```bash
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

3. Restart the dev server: `npm run dev`

The AI Tutor will now use Claude for real Socratic responses instead of the local fallback engine.

## Chrome Extension — Lifeline Recall

The extension works on **any webpage** including YouTube, UWorld, Amboss, textbooks, and web articles.

### Install
1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked** → select the `extension/` folder
4. Pin the 🟢 L icon to your toolbar

### Add Anthropic API Key to Extension
1. Click the L icon in Chrome
2. In the "Anthropic API Key" field, paste your key
3. Click **Save API Key**

Cards are now generated by Claude — with Step 1 and Step 2 relevant content added automatically.

### How to use
- **On any page**: Highlight 30+ characters of text → floating "Capture with Lifeline" button appears → click → sidebar opens → Generate
- **On YouTube**: A Lifeline bar appears below the video → capture current subtitle, last 60s, or full transcript
- **Right-click**: Select text → right-click → "Capture with Lifeline Recall"
- **Edit any card**: Click ✏️ Edit → a full-screen modal opens (fully editable)

## Full Stack Setup (Optional)

For the FastAPI backend with real database:

### Prerequisites
- Python 3.11+
- PostgreSQL 15+
- Redis 7+

### Backend Setup

```bash
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Create database
createdb lifeline_meded
psql -d lifeline_meded -f ../database/schema.sql

# Configure environment
cp ../.env.example .env
# Edit .env with your values

# Start backend
uvicorn main:app --reload --port 8000
# API docs at http://localhost:8000/docs
```
